# BillboardIQ - Digital Marketing Services SPA

A modern Single Page Application (SPA) built with React.js for BillboardIQ, a company offering Digital Marketing services and Retargeting Campaigns for Independent Billboard Operators.

## Features

- **Modern React SPA**: Built with React 18 and modern hooks
- **Responsive Design**: Fully responsive layout that works on all devices
- **Professional UI**: Beautiful, modern interface with smooth animations
- **Key Sections**:
  - Hero section with company branding
  - Services overview
  - How It Works (4-step process)
  - Benefits and statistics
  - Metrics tracking information
  - Call-to-action section
  - Footer with links

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
